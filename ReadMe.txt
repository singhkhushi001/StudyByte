StudyByte - Online Course Provider
Overview
StudyByte is a simple online course provider platform built with HTML, CSS, and JavaScript. It consists of three pages:

Home Page: Introduces the platform, highlights its key features, and provides a CTA to browse available courses.
Courses Page: Displays a list of available courses with options to enroll.
Contact Page: Contains a contact form that allows users to send inquiries. The form includes basic validation using JavaScript.
Features
Responsive Design: Works across different screen sizes and devices.
Form Validation: The contact form includes basic client-side validation for required fields.
Dynamic Interaction: Simple JavaScript enhances user interaction on the contact page.

Pages Overview
Home Page (index.html):

A hero section introducing the platform.
Features section highlighting the key benefits of using StudyByte.
CTA button linking to the Courses page.
Courses Page (courses.html):

A list of available courses such as Web Development, Data Science, and Graphic Design.
Each course section contains a button for enrollment.
Contact Page (contact.html):

A contact form where users can enter their name, email, and message.
Includes basic form validation with an alert system to notify users when a message is sent or when the form is incomplete.
Setup Instructions
Download the Project: Clone or download this repository to your local machine.

Open in a Browser:

Open the index.html file in a web browser to view the homepage.
Navigate through the different pages by clicking the links in the navigation bar.
Customization
To add new courses, modify the courses.html file and add new sections to the course grid.
You can modify the form validation logic in the script.js file to add more complex validation if necessary.
